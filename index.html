YouTube Notes Privacy Policy

YouTube Notes stores user-created notes locally in the browser using Chrome storage APIs.

The extension does not collect, transmit, sell, or share personal data with third parties.

The extension only accesses YouTube video pages in order to:

  • Save timestamped notes
  • Display note markers on videos
  • Allow users to revisit saved notes

All note data remains on the user's device unless the user manually exports it.

No analytics, tracking, advertising, or external servers are used.
